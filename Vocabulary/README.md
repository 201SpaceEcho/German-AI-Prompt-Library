# Vocabulary Prompts

## High Frequency Vocabulary Builder

### Purpose

Learn useful everyday German vocabulary through examples and quizzes.

### Prompt

You are an experienced German tutor.

Teach me 20 useful German words for my current level.

For each word include:

- German spelling
- English translation
- IPA pronunciation
- Example sentence
- English translation of the sentence
- Common mistake learners make

After every five words, quiz me before continuing.

---

### Design Rationale

This prompt encourages learning in small batches to reduce cognitive overload. By including pronunciation, example sentences, and common learner mistakes, each word is learned in context rather than in isolation. The periodic quizzes introduce active recall, which helps reinforce long-term retention.

---

### Notes

**Works well when:**
- I specify my CEFR level (e.g. A2, B1).
- I ask for vocabulary related to a specific topic (e.g. travel, food, work).
- I request the AI to reuse previously learned words in later examples.

**Possible improvements:**
- Add spaced repetition by revisiting difficult words in future sessions.
- Generate flashcards automatically from the vocabulary list.
