# AI Experiments / Prompt Testing

## Overview

This section documents experiments with different AI prompt structures for German language learning.

The goal is to evaluate how prompt design affects learning outcomes, clarity, and user engagement.

---

## Experiment 1: Short vs Detailed Corrections

### Question

Do learners benefit more from short corrections or detailed grammatical explanations?

### Setup

I tested two prompt styles:

**A. Minimal correction style**
- Only corrects mistakes
- No explanations

**B. Teaching correction style**
- Corrects mistakes
- Explains grammar rules
- Provides alternative phrasing

### Observation

- Style A is faster but less educational
- Style B improves understanding but slows conversation
- Best results come from combining both depending on learner level

---

## Experiment 2: CEFR Level Control

### Question

Does specifying CEFR level improve output quality?

### Observation

Yes. Without CEFR levels, responses are often too advanced.

When specifying A2/B1:
- Vocabulary becomes more controlled
- Sentence structure becomes more appropriate
- Explanations are clearer

---

## Experiment 3: Output Length Constraints

### Question

Does limiting AI response length improve conversation quality?

### Observation

Shorter responses (2–4 sentences):
- increase interaction frequency
- improve learner participation
- reduce cognitive overload
