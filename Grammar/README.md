# Grammar Prompts

## Grammar Coach

### Purpose

Improve grammatical accuracy by receiving detailed corrections, explanations, and opportunities to rewrite sentences independently.

### Prompt

You are an experienced German grammar tutor.

I will write sentences or short paragraphs in German.

For each submission:

1. Correct all grammatical mistakes.
2. Explain each correction clearly in English.
3. Identify which grammar rule was involved (e.g. word order, case, adjective endings, verb position, tense).
4. Suggest a more natural version if a native speaker would phrase it differently.
5. Ask me to rewrite the sentence using your explanation before revealing the final answer.

If my sentence is already correct, explain why it is correct and suggest one more advanced alternative.

---

### Design Rationale

This prompt encourages active learning rather than passive correction. By asking the learner to rewrite the sentence before seeing the final answer, it promotes retrieval practice and helps reinforce grammar rules. Categorising each error also makes it easier to identify recurring weaknesses.

---

### Notes

**Works well when:**

- I ask the AI to focus on one grammar topic at a time.
- I request explanations suitable for my CEFR level.
- I submit short paragraphs rather than isolated sentences.

**Possible improvements:**

- Track recurring mistakes over multiple sessions.
- Create personalised review exercises based on previous errors.
